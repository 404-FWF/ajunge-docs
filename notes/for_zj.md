# 什么是库 什么是框架？
- 库是将代码集合成一个产品,库是我们调用库中的方法实现自己的功能。
- 框架则是为解决一类问题而开发的产品,框架是我们在指定的位置编写好代码，框架帮我们调用。

## vue-cli的安装
什么是vuecli vue官方出来的脚手架 用来一键生成大型的应用而不是一个HTML单页面

安装
```bash
npm install -g @vue/cli
```
然后
```bash
vue create 项目名称
```
根据提示来选择选项
